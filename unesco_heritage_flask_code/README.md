# UNESCO World Heritage Sites Analysis

A professional Flask website showcasing UNESCO World Heritage Sites data analysis through Tableau visualizations.

## Design Theme

- **Primary Colors**: Heritage Teal (#1a6b5c) & Deep Teal (#0d4a3d)
- **Accent Colors**: Heritage Gold (#c9a227) & Gold Light (#e8c84a)
- **Background**: Warm Cream (#faf8f3)
- **Typography**: Cormorant Garamond (headings) + Inter (body)

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Open `http://localhost:5000` in your browser.
